# 3D Printable Cases

These are 3D printable cases for:
- Housing both the ESP32 dev module and the S3 card module (both without header pins) in a single case. **Note:** This is a very basic case designed just to keep the modules inside, and it might not fit the specific dev board you are using.
- There is a case for a 0.91 inch OLED screen as well

**Note:** The cap of the case needs a bit of glue to keep it secure.
